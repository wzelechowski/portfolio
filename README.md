## 🌟 Featured Projects

### 🍕 PolishPizza
A full-scale microservices-based pizzeria management system built with Spring Boot and Spring Cloud. It includes API Gateway, service discovery (Eureka), centralized configuration, and asynchronous communication via RabbitMQ. The system uses PostgreSQL and Redis and provides both a React admin panel and a mobile client. It also integrates an AI module for personalized promotion generation.  
🔗 https://github.com/wzelechowski/PolishPizza

---

### ⚡ Sportradar_BE
A high-performance Spring Boot backend focused on advanced PostgreSQL query optimization. Implements solutions such as N+1 query elimination (LEFT JOIN FETCH), batch fetching to avoid Cartesian product issues, and dynamic filtering and pagination using Criteria API. Fully containerized with Docker and paired with a Vanilla JS frontend.  
🔗 https://github.com/wzelechowski/sportradar_BE

---

### 🩺 Healthy
A distributed IoT system for real-time monitoring of patients’ vital signs. The system processes streaming data and applies machine learning models for anomaly detection and health analysis. It uses InfluxDB for time-series data storage and was developed as a multi-service architecture project.  
🔗 https://github.com/247659/Healthy
