# Patients Service

## Opis
Usługa odpowiedzialna za zarządzanie kontami, profilami i danymi demograficznymi samych pacjentów. 

## Odpowiedzialności
* **Zarządzanie profilem:** Przechowywanie zanonimizowanych danych i preferencji pacjentów, zgodnie z wymogami RODO.
* **Konfiguracja progów:** Przechowywanie indywidualnych progów alarmowych dla konkretnych pacjentów.
* **Obsługa aplikacji mobilnej:** Dostarczanie danych profilowych dla interfejsu React Native.

## Technologie
* Spring Boot
* PostgreSQL
* Docker
