# Medical Staff Service

## Opis
Serwis obsługujący logikę biznesową dla personelu medycznego korzystającego z panelu webowego.

## Odpowiedzialności
* **Zarządzanie przypisaniami:** Obsługa relacji między lekarzami a pacjentami.
* **Pulpit lekarza:** Agregacja danych niezbędnych do wyświetlenia listy monitorowanych pacjentów.
* **Notatki kliniczne:** Umożliwienie personelowi dodawania notatek medycznych do konkretnych zdarzeń i alertów.

## Technologie
* Spring Boot
* PostgreSQL
* Docker
